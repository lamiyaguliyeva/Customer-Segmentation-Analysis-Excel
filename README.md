# Customer-Segmentation-Analysis-Excel
Customer segmentation analysis using Excel, Power Query, Power Pivot and Pivot Tables to generate business insights.

## Tools Used
- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables
- Data Visualization

## Project Overview
This project focuses on customer segmentation analysis using Excel. Data was cleaned and transformed with Power Query, modeled with Power Pivot, and analyzed through Pivot Tables to identify customer groups and generate business insights.
